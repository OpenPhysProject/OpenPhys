OpenPhys Interactive Physics Simulations
This site contains a variety of learning objects featuring interactive simulations covering modern physics topics at approximately a 100 and 200 university level. The material is formatted to allow learners to freely explore and dynamically adjust the content level. This site is an HTML5 web app designed to support all modern devices including smart phones, tablets, and desktops. This project is open source and freely available on GitHub https://github.com/OpenPhysProject/OpenPhys
