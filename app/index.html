Interactive Physics
This guided self-study site contains a set of modern physics modules at a 100 to 200 university level. Each module is based on an innovative game-like navigable 2D map providing a conceptual framework. Explore the territory at your own level and own pace (see User's Guide -->).

The site has been specifically designed for both desktop and mobile use. This project is open source and freely available here: https://github.com/OpenPhysProject/OpenPhys . The map framework is easily reusable for any type of content.
